# ZenStack SaaS Backend Template

SaaS Backend Template using express.js

## Features

-   Multi-tenant
-   Soft delete
-   Sharing by groups

## Running

1. Install dependencies

```bash
npm install
```

2. build

```bash
npm run build
```

3. seed data

```bash
npm run seed
```

4. start

```bash
npm run dev
```

5. test

```bash
curl -H "X-USER-ID:alice@prisma.io" localhost:3000/post
```
